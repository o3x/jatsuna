# 蛇突奈 (Jatsuna) - React Refactor

3人対戦型戦略ボードゲーム「蛇突奈 (Jatsuna)」のReactリファクタリング版です。
オセロ（リバーシ）をベースに、3つの勢力（オーブ、ジェム、ステラ）が盤面を奪い合います。

## 特徴

- **3人対戦**: 通常のオセロとは異なる、三つ巴の戦略が必要となります。
- **多彩なAI**: EasyからSuperHard、そしてプレイヤーを陥れる「最凶結託モード」まで搭載。
- **モダンなUI**: React + Tailwind CSSによる洗練されたデザインとアニメーション。

## セットアップ

このプロジェクトを実行するには、Node.jsが必要です。

1. 依存関係のインストール:
   ```bash
   npm install
   ```

2. 開発サーバーの起動:
   ```bash
   npm run dev
   ```

3. ビルド:
   ```bash
   npm run build
   ```

## ゲームルール

- **基本**: 自分の石で相手の石を挟むと、自分の色に変わります。
- **3すくみ**: 3プレイヤー（O, C, P）が順番に石を置きます。
- **勝利条件**: 盤面が埋まるか、全員がパスした時点で、最も多くの石を持っているプレイヤーの勝利です。
- **特殊ルール**:
    - **結託モード**: AI 2人が結託して、あなた（プレイヤー）を全力で潰しに来ます。

## 技術スタック

- React 18
- Vite
- Tailwind CSS
